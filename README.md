# AICode CLI

AI-powered CLI code generation tool.

## Installation

```bash
npm install -g aicode
```

## Usage

```bash
# Show version
aicode --version

# Show help
aicode --help

# Run with custom config
aicode --config ./aicode.config.json

# Enable debug mode
aicode --debug

# Enable verbose output
aicode --verbose
```

## Development

```bash
# Install dependencies
npm install

# Build
npm run build

# Run tests
npm test

# Run tests with coverage
npm run test:coverage

# Run linter
npm run lint

# Watch mode
npm run dev
```
